The identity of a builder.

A design system for Bruce Peters. The mark is his initials notated as a phoneme — /bp/ — the same color-coded sound language he built into Word Wiz, set as a code token and dropped into a terminal world. Personal, engineered, and impossible to mistake for anyone else.

If a human would say it, set it in Bricolage. If a machine would print it — a label, a path, a stat, a timestamp, a tag — set it in mono. The tension between the two is the whole personality: warm voice, engineered chrome.

05

Voice

Plainspoken, present-tense, lowercase. A builder talking about what he's building — not a landing page selling a vision.

✓ sounds like bruce

"i build things that outlast me."

"my code scored every point."

"a reading tutor that hears the sounds kids miss."

  • Lead with the verb. Build, ship, score, hear.
  • Concrete > grand. Name the real thing.
  • Lowercase headers feel like a commit message.
✕ too much

"Infrastructure for human potential."

"Leveraging cutting-edge AI to transform outcomes."

"A passionate, results-driven engineer."

  • No mission-statement abstraction.
  • No résumé adjectives — show the work instead.
  • No hype verbs: leverage, transform, revolutionize.
